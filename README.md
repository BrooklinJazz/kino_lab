## Installation

Add `:kino_lab` as a dependency in your notebook:

```elixir
Mix.install([
  {:kino, "~> 0.5.1"},
  {:kino_lab, "~> 0.1.0-dev", github: "jonatanklosko/kino_lab"}
])
```
